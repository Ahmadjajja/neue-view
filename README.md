## Setup 

1. Install Python 3.9 or later.

2. Clone this repository:

    ```bash
    git clone https://github.com/Louvivien/nueu-view.git 
    cd nueu-view
    ```


Open another terminal windows
    
3. 
    ```bash

cd client
    ```


4. install packages
    ```bash

   npm i
    ```


5. start the client
    ```bash

   npm start

    ```
